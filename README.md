# Platform

> any linux platform
> but test on Fedora 34+

# Usage

> install by running the following command

```sh
chmod +x install
./install
```

# Keymap

> <Ctrl-Shift-v> -> 粘贴选中的内容

> <Alt-v> -> 垂直分屏

> <Alt-h> -> 水平分屏

| --- | --- |
| <Ctrl-Shift>H | 光标跳转到左边的屏幕 |
| <Ctrl-Shift>J | 光标跳转到下边的屏幕 |
| <Ctrl-Shift>K | 光标跳转到上边的屏幕 |
| <Ctrl-Shift>L | 光标跳转到右边的屏幕 |
